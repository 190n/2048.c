2048.c
======

![screenshot](http://www.leaseweblabs.com/wp-content/uploads/2014/03/2048.png)

Console version of the game "2048" for GNU/Linux

### Requirements

- gcc

### Installation

```
wget https://raw.githubusercontent.com/mevdschee/2048.c/master/2048.c
gcc -o 2048 2048.c
./2048
```

### Background Colors

Blue: 
`./2048 -b`

Bright Red:
`./2048 -r`

Green:
`./2048 -g`

Cyan:
`./2048 -c`

Purple:
`./2048 -p`

Yellow:
`./2048 -y`
